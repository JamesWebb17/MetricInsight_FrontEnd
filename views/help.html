MetricInsight help page

How to use MetricInsight ?

First, you need to configure the program. You can do it by clicking on the "Configuration" button in the navigation bar.

Then, you can start the program by clicking on the "Start" button in the navigation bar.

Finally, you can see the results of the program by clicking on the "Results" button in the navigation bar.

How to configure MetricInsight ?

First, you need to choose the metrics you want to collect. You can do it by checking the boxes corresponding to the metrics you want to collect.

Then, you need to choose the time interval between each collection. You can do it by choosing a value in the "Time interval" field.

Finally, you need to choose the duration of the program. You can do it by choosing a value in the "Duration" field.

Then, you can start the program by clicking on the "Start" button in the navigation bar.

Finally, you can see the results of the program by clicking on the "Results" button in the navigation bar.
